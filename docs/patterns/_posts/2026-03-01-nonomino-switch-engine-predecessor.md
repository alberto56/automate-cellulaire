    Charles Corderman may have found this object in 1971 while
    exhaustively investigating the [fate](/patterns/fate) of all the small [polyomino](/patterns/polyomino)es.
    Records indicate that he found the [switch engine](/patterns/switch-engine) while
    investigating the decominoes (polyominoes having 10 cells). However,
    there do not appear to be decominoes which result in a [clean](/patterns/clean)
    [switch engine](/patterns/switch-engine).  If Corderman was examining polyominoes in order of
    size, then this smaller [predecessor](/patterns/predecessor) should have been found first in
