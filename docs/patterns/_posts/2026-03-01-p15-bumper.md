description: |
  A periodic [colour-preserving](/patterns/colour-preserving) [glider](/patterns/glider) [reflector](/patterns/reflector) with
  [Karel's p15](/patterns/karel-s-p15) providing the necessary [spark](/patterns/spark).  The minimum
  [repeat time](/patterns/repeat-time) is 45 ticks.  For an equivalent [colour-changing](/patterns/colour-changing)
  periodic glider reflector see [p15 bouncer](/patterns/p15-bouncer).  A [stable](/patterns/stable) [Snark](/patterns/snark)
  reflector can be substituted for any [bumper](/patterns/bumper).  This changes the
  timing of the output glider, which can be useful for rephasing
  periodic glider streams.
