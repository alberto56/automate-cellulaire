description: |
  Any pattern whose [population](/patterns/population) grows at a quadratic rate,
  although it is usual to exclude [spacefiller](/patterns/spacefiller)s.  It is easy to see
  that this is the fastest possible growth rate.
  The term is also sometimes used to mean specifically the breeder
  created by Bill Gosper's group at MIT, which was the first known
  pattern exhibiting [superlinear growth](/patterns/superlinear-growth).
  There are four common types of breeder, known as MMM, MMS, MSM and
  SMM (where M=moving and S=stationary).  Typically an MMM breeder is a
  [rake](/patterns/rake) [puffer](/patterns/puffer), an MMS breeder is a puffer producing puffers which
  produce stationary objects ([still life](/patterns/still-life)s and/or [oscillator](/patterns/oscillator)s), an
  MSM breeder is a [gun](/patterns/gun) puffer and an SMM breeder is a rake gun. There
  are, however, less obvious variants of these types.  Other less
  common breeder categories (SSS, hybrid MSS/MSM, etc.) can be created
  with some difficulty, based on [universal constructor](/patterns/universal-constructor) technology;
  see [Pianola breeder](/patterns/pianola-breeder).
  The original breeder was of type MSM (a p64 puffer puffing p30
  glider guns).  The known breeder with the smallest initial population
  is [switch-engine ping-pong](/patterns/switch-engine-ping-pong).
