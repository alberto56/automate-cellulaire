description: |
  A certain reaction between a block and a
  [Herschel](/patterns/herschel) [predecessor](/patterns/predecessor) in which the block reappears in its original
  place some time later, the reaction having effectively passed through
  it.  This reaction was found by Dave Buckingham in 1988.  It has been
  used in some [Herschel conduit](/patterns/herschel-conduit)s, and in the [gunstar](/patterns/gunstar)s.  Because the
  reaction involves a Herschel predecessor rather than an actual
  Herschel, the following diagram shows instead a [B-heptomino](/patterns/b-heptomino) (which
  by itself would evolve into a block and a Herschel).
