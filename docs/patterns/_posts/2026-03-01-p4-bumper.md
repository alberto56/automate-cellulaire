description: |
  (p4)  A periodic [colour-preserving](/patterns/colour-preserving) [glider](/patterns/glider) [reflector](/patterns/reflector)
  with a minimum [repeat time](/patterns/repeat-time) of 36.  Unlike the p5 through p8 cases
  where Noam Elkies' [domino](/patterns/domino) spark-based reflectors are available, no
  small period-4 [colour-changing](/patterns/colour-changing) reflector is known.  A [stable](/patterns/stable)
  [Snark](/patterns/snark) reflector can be substituted for any [bumper](/patterns/bumper).  This changes
  the timing of the output glider, which can be useful for rephasing
  periodic glider streams.
