description: |
  Opposite of [dirty](/patterns/dirty).  A reaction which produces a small number
  of different products which are desired or which are easily deleted
  is said to be clean.  For example, a [puffer](/patterns/puffer) which produces just one
  object per period is clean.  Clean reactions are useful because they
  can be used as building blocks in larger constructions.
  When a [fuse](/patterns/fuse) is said to be clean, or to [burn](/patterns/burn) cleanly, this
  usually means that no debris at all is left behind.
