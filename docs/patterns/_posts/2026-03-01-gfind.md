description: |
  A program by David Eppstein which uses [de Bruijn graph](/patterns/de-bruijn-graph)s to
  search for new [spaceship](/patterns/spaceship)s.  It was with gfind that Eppstein found
  the [weekender](/patterns/weekender), and Paul Tooke later used it to find the [dragon](/patterns/dragon).
  It is available at [http://www.ics.uci.edu/~eppstein/ca/gfind.c](http://www.ics.uci.edu/~eppstein/ca/gfind.c) (C
  source code only).
  Compare [lifesrc](/patterns/lifesrc).
