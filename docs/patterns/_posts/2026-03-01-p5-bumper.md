description: |
  A periodic [colour-preserving](/patterns/colour-preserving) [glider](/patterns/glider) [reflector](/patterns/reflector) with a
  [middleweight volcano](/patterns/middleweight-volcano) producing the necessary [spark](/patterns/spark).  The minimum
  [repeat time](/patterns/repeat-time) is 35 ticks.  For an equivalent [colour-changing](/patterns/colour-changing)
  periodic glider reflector see [p5 bouncer](/patterns/p5-bouncer).  A [stable](/patterns/stable) [Snark](/patterns/snark)
  reflector can be substituted for any [bumper](/patterns/bumper).  This changes the
  timing of the output glider, which can be useful for rephasing
  periodic glider streams.
