description: |
  A period-multiplying [colour-preserving](/patterns/colour-preserving) [signal](/patterns/signal)
  [conduit](/patterns/conduit) found by Tanner Jacobi in October 2017, producing one
  output [glider](/patterns/glider) for every two input gliders.  It is made by replacing
  one of the eaters in a [Snark](/patterns/snark) with a [catalyst](/patterns/catalyst) found using
  [Bellman](/patterns/bellman).  The catalyst causes the formation of a [tub](/patterns/tub) which
  requires a second glider to delete.  However, this adds 5 ticks to
  the repeat time, so that it becomes 48.  This is still 3 ticks faster
  than the [CC semi-Snark](/patterns/cc-semi-snark).
