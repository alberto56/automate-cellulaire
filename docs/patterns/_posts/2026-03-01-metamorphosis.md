description: |
  An [oscillator](/patterns/oscillator) built by Robert Wainwright that uses
  the following reaction (found by Bill Gosper) to turn [glider](/patterns/glider)s into
  [LWSS](/patterns/lwss), and converts these LWSS back into gliders by colliding them
  head on using an [LWSS-LWSS bounce](/patterns/lwss-lwss-bounce).  There are two ways to do the
  following reaction, because the [twin bees shuttle spark](/patterns/twin-bees-shuttle-spark) is
  [symmetric](/patterns/symmetric).
