description: |
  A [slow salvo](/patterns/slow-salvo) that uses gliders of only one
  colour.  For example, the slow salvos generated by
  [half-baked knightship](/patterns/half-baked-knightship)s are monochromatic, because they are
  generated by a single type of reaction which can happen at any
  position along a diagonal line.  The smallest possible step size is
  one [full diagonal](/patterns/full-diagonal) (1fd), which is two [half diagonal](/patterns/half-diagonal)s (2hd), which
  means that any single glider-producing reaction can only reach half
  of the available glider [lane](/patterns/lane)s.  See [colour of a glider](/patterns/colour-of-a-glider).
