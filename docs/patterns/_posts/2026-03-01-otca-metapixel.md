description: |
  (p46 circuitry)  A 2048 x 2048 period 35328 [metacell](/patterns/metacell)
  constructed by Brice Due in 2006.  It contains a large "pixel" area
  that contains a large population of [LWSS](/patterns/lwSs)es when the metacell state
  is ON, but is empty when it is OFF.  This allows the state of the
  metacell to be visible at high zoom levels, unlike previous
  [unit cell](/patterns/unit-cell)s where the state was signaled by the presence or absence
  of a single glider in a specific location.
