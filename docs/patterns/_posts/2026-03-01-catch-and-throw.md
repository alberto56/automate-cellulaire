description: |
  A [technology](/patterns/technology) used (e.g., in the [Caterpillar](/patterns/caterpillar)) to
  adjust the timing of a glider by turning it into a stationary object
  using one interaction, and then later restoring it using a second
  interaction.  The interactions are caused by passing objects which
  are not otherwise affected.  The direction of the glider is not
  usually changed.
  Here is an example where a glider is turned into a [boat](/patterns/boat) by the
  first [LWSS](/patterns/lwSs), and is then restored by the remaining [spaceship](/patterns/spaceship)s.
