description: |
  Any reaction in which a [glider](/patterns/glider) is turned onto a new
  path by a [spaceship](/patterns/spaceship), [oscillator](/patterns/oscillator), or [still life](/patterns/still-life) [constellation](/patterns/constellation).
  In the last two cases, the glider turner is usually called a
  [reflector](/patterns/reflector) if the reaction is repeatable, or a [one-time](/patterns/one-time) [turner](/patterns/turner)
  if the reaction can only happen once.
  Glider turners are easily built using [standard spaceship](/patterns/standard-spaceship)s. The
  following diagram shows a convoy which turns a [forward glider](/patterns/forward-glider) 90
  degrees, with the new glider also moving forwards.
