description: |
  A type of test reaction in [memory cell](/patterns/memory-cell)
  circuitry, where the information in the memory cell is unchanged and
  can be read again to produce the same result.  One simple type of
  non-destructive read reaction is a [signal](/patterns/signal) sent to a
  [permanent switch](/patterns/permanent-switch).  Memory cells with [destructive read](/patterns/destructive-read) reactions
  are generally simpler and more commonly used.
