description: |
  A [Spartan](/patterns/spartan) [converter](/patterns/converter) found by Tanner Jacobi in October
  2015, which converts an input [Herschel](/patterns/herschel) to a middleweight spaceship.
  The key discovery was a very small but slightly [dirty](/patterns/dirty) H-to-MWSS
  conduit, where a Herschel is catalyzed to produce an [MWSS](/patterns/mwSs) but also
  leaves behind a beehive.  Prefixing two [R64](/patterns/r64) conduits to this
  produces a [composite](/patterns/composite) converter that successfully deletes the
  beehive in advance, using the input Herschel's
  [first natural glider](/patterns/first-natural-glider).
