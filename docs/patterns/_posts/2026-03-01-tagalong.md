description: |
  An object which is not a [spaceship](/patterns/spaceship) in its own right, but
  which can be attached to one or more spaceships to form a larger
  spaceship.  For examples see [Canada goose](/patterns/canada-goose), [fly](/patterns/fly), [pushalong](/patterns/pushalong),
  [sidecar](/patterns/sidecar) and [sparky](/patterns/sparky).  See also [Schick engine](/patterns/schick-engine), which consists of
  a tagalong attached to two LWSS (or similar).
  The following [c/4 spaceship](/patterns/c/4-spaceship) (Nicolay Beluchenko, February 2004)
  has two wings, either of which can be considered as a tagalong. But
  if either wing is removed, then the remaining wing becomes an
  essential component of the spaceship, and so is no longer a tagalong.
