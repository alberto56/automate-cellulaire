description: |
  An alternative name for a [boat-bit](/patterns/boat-bit).  Not a very sensible
  name, because various other things can be used instead of a snake.  A
  snake, or alternatively an [aircraft carrier](/patterns/aircraft-carrier), is the smallest object
  that can consume a glider [stream](/patterns/stream) by effectively acting as an
  [eater](/patterns/eater) for every two incoming gliders.  The one-cell reduction from
  the smallest real eater, the seven-cell [eater1](/patterns/eater1), has been important
  when trying to construct recent [sawtooth](/patterns/sawtooth)s where the [population](/patterns/population)
  must be minimized.
