description: |
  A period-multiplying [colour-preserving](/patterns/colour-preserving) [signal](/patterns/signal)
  [conduit](/patterns/conduit) found by Tanner Jacobi in October 2017, producing one
  output [glider](/patterns/glider) for every four input gliders.  It is made by
  replacing one of the eaters in a [tremi-Snark](/patterns/tremi-snark) with a [catalyst](/patterns/catalyst)
  found using [Bellman](/patterns/bellman).  The catalyst causes the formation of a [tub](/patterns/tub)
  which then requires an additional glider to delete.  However, this
  adds 5 ticks to the repeat time, so that it becomes 48.  If period
  quadrupling is needed with a [colour-changing](/patterns/colour-changing) reaction, a
  [CP semi-Snark](/patterns/cp-semi-snark) and a [CC semi-Snark](/patterns/cc-semi-snark) can be used in series, or a
  period-multiplying [Herschel conduit](/patterns/herschel-conduit) can be connected to a [syringe](/patterns/syringe)
  and an appropriately chosen Herschel-to-glider [converter](/patterns/converter).
