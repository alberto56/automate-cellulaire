description: |
  A periodic [colour-preserving](/patterns/colour-preserving) [glider](/patterns/glider) [reflector](/patterns/reflector) with a
  [blocker](/patterns/blocker) attached to provide the necessary spark.  The minimum
  [repeat time](/patterns/repeat-time) is 40 ticks.  For an equivalent [colour-changing](/patterns/colour-changing)
  periodic glider reflector see [p8 bouncer](/patterns/p8-bouncer).  A [stable](/patterns/stable) [Snark](/patterns/snark)
  reflector can be substituted for any [bumper](/patterns/bumper).  This changes the
  timing of the output glider, which can be useful for rephasing
  periodic glider streams.
