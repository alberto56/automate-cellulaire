  The following reaction, found by David Bell in 1996, in
  which two [glider](/patterns/glider)s appear to circle around each other as they are
  reflected 90 degrees by a [twin bees shuttle](/patterns/twin-bees-shuttle).  Four copies of the
  reaction can be used to create a p92 glider loop which repeats the
  do-see-do reaction forever.
