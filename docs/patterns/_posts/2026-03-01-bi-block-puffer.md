description: |
  Any [puffer](/patterns/puffer) whose output is [bi-block](/patterns/bi-block)s.  The term
  is particularly used for p8 c/2 puffers, in which case a
  [bi-block fuse](/patterns/bi-block-fuse) is created.  A bi-block puffer is easily made using
  two [backrake](/patterns/backrake)s whose gliders impact symmetrically.  Jason Summers
  [weld](/patterns/weld)ed two backrakes to form a more compact puffer, as shown below. By periodically burning the [bi-block fuse](/patterns/bi-block-fuse) using perturbations by a
  following backrake and spaceships, c/2 rakes can be created for all
  periods that are a multiple of eight.
