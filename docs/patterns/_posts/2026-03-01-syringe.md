description: |
  A small stable [converter](/patterns/converter) found by Tanner Jacobi in March
  2015, accepting a glider as input and producing an output [Herschel](/patterns/herschel)
  As of June 2018 it is the smallest known converter of this type, so
  it is very often used to handle input gliders in complex [signal](/patterns/signal)
  [circuit](/patterns/circuit)ry, as described in [Herschel circuit](/patterns/herschel-circuit).  A second glider can
  safely follow the first any time after 78 ticks, but [overclocking](/patterns/overclocking)
  also allows the syringe to work at a [repeat time](/patterns/repeat-time) of 74 or 75 ticks.
  If followed by a [dependent conduit](/patterns/dependent-conduit) a simple [eater2](/patterns/eater2) can be used
  instead of the large [weld](/patterns/weld)ed [catalyst](/patterns/catalyst) shown here.  A
  [ghost Herschel](/patterns/ghost-herschel) marks the output location.
