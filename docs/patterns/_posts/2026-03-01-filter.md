description: |
  Any [oscillator](/patterns/oscillator) used to delete some but not all of the
  [spaceship](/patterns/spaceship)s in a stream.  An example is the [blocker](/patterns/blocker), which can be
  positioned so as to delete every other [glider](/patterns/glider) in a stream of period
  8n+4, and can also do the same for [LWSS](/patterns/lwSs) streams.  Other examples
  are the [MW emulator](/patterns/mw-emulator) and [T-nosed p4](/patterns/t-nosed-p4) (either of which can be used
  to delete every other LWSS in a stream of period 4n+2), the
  [fountain](/patterns/fountain) (which does the same for [MWSS](/patterns/mwSs) streams) and a number of
  others, such as the p6 [pipsquirter](/patterns/pipsquirter), the [pentadecathlon](/patterns/pentadecathlon) and the
  p72 oscillator shown under [factory](/patterns/factory).  Another example, a p4
  oscillator deleting every other HWSS in a stream of period 4n+2, is
