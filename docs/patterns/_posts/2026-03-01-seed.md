description: |
  A [constellation](/patterns/constellation) of still lifes and/or oscillators, which can
  be converted into another Life object when it is struck by one or
  more gliders.  Usually the resulting object is a rare still life or
  spaceship, more complex than the original constellation.  [Spartan](/patterns/spartan)
  single-glider (1G) seeds are more commonly seen than multi-glider
  seeds, because a Spartan 1G seed can be readily constructed and
  [trigger](/patterns/trigger)ed using a [slow salvo](/patterns/slow-salvo).  See also [freeze-dried](/patterns/freeze-dried).  For
  example, the following is a 14[sL](/patterns/sl) 1G seed for a c/7 loafer
  spaceship.
