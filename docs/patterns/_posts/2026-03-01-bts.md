  A 19-cell [still life](/patterns/still-life) made up of a [bookend](/patterns/bookend), a [table](/patterns/table), and a
  [snake](/patterns/snake).  Starting in 2015, with the help of Mike Playle's [Bellman](/patterns/bellman)
  search program, Tanner Jacobi discovered a surprising number of ways
  to use this object as a [catalyst](/patterns/catalyst) for [signal](/patterns/signal) [circuit](/patterns/circuit)ry.  One
  example can be seen in the [CC semi-cenark](/patterns/cc-semi-cenark) entry.
