description: |
  A [conduit](/patterns/conduit) with no recognizable active signal
  stage besides its input and output.  An early example still very
  commonly used is Buckingham's [BFx59H](/patterns/bfx59h), which transforms a
  [B-heptomino](/patterns/b-heptomino) into an inverted [Herschel](/patterns/herschel) in 59 ticks.  The BFx59H
  elementary conduit is a component in many of the original [universal](/patterns/universal)
  [toolkit](/patterns/toolkit) of Herschel conduits.  An extension of the same naming
  convention is used for elementary conduits, with the first and last
  letters of the name specifying the input and output [signal](/patterns/signal) objects.
  As with Herschels, an arbitrary orientation and center point is
  chosen for each object.  "Fx" means the signal moves forward and
  produces a mirror-image output.  See [Herschel conduit](/patterns/herschel-conduit) for further
  details.
  Theoretically an elementary conduit may become a composite conduit,
  if another conduit can be found that shares the beginning or end of
  the conduit in question.  In practice this happens only rarely,
  because many of the most likely branch points have already been
  identified: [glider](/patterns/glider) (G), [LWSS](/patterns/lwSs) (L) or [MWSS](/patterns/mwSs) (M), [Herschel](/patterns/herschel) (H),
  [B-heptomino](/patterns/b-heptomino) (B), [R-pentomino](/patterns/r-pentomino) (R), [pi-heptomino](/patterns/pi-heptomino) (P),
  [queen bee shuttle](/patterns/queen-bee-shuttle) (Q), [century](/patterns/century) or [bookend](/patterns/bookend) (C), [dove](/patterns/dove) (D), and
  [wing](/patterns/wing) (W).  A [Herschel descendant](/patterns/herschel-descendant) might qualify, due to the
  elementary conduit that can be seen in the [p184 gun](/patterns/p184-gun).  However,
  there are very few simple conduits that produce Herschel descendants
  without Herschels, so in practice this is not a useful branch point.
