description: |
  A specific three-[tick](/patterns/tick) predecessor of a
  [Herschel](/patterns/herschel), commonly seen in [Herschel conduit](/patterns/herschel-conduit) collections that
  contain [dependent conduit](/patterns/dependent-conduit)s.  In some situations it is helpful to
  display the input reaction in this form instead of the standard
  Herschel form.

  description: |
    Dependent conduit inputs are catalysed by a [transparent](/patterns/transparent) block
    before the Herschel's standard form can appear, and before the
    Herschel's [first natural glider](/patterns/first-natural-glider) is produced.  This means that these
    conduits will fail if an actual Herschel is placed in the "correct"
    input location for a dependent conduit.  Refer to [F166](/patterns/f166) or [Lx200](/patterns/lx200)
    to see the correct relative placement of the standard transparent
    block catalyst.
    Almost all known Herschel conduits produce a Herschel
    great-grandparent near the end of their evolutionary sequence.  In
    the original [universal](/patterns/universal) set of Herschel conduits, [Fx158](/patterns/fx158) is the
    only exception.
