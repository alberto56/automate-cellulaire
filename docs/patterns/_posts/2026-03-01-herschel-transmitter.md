description: |
  Any [Herschel](/patterns/herschel)-to-two-[glider](/patterns/glider) [converter](/patterns/converter) that
  produces a [tandem glider](/patterns/tandem-glider) that can be used as input to a
  [Herschel receiver](/patterns/herschel-receiver).  If the gliders are far enough apart, and if one
  of the gliders is used only for cleanup, then the transmitter is
  [ambidextrous](/patterns/ambidextrous):  with a small modification to the receiver, a
  suitably oriented mirror image of the receiver will also work.
  The following diagram shows a [stable](/patterns/stable) Herschel transmitter found
  by Paul Callahan in May 1997.
